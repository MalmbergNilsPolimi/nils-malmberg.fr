Le site nils-malmberg.fr

Contexte

Durant ma première année d'école d'ingénieur, j'ai commencé à développer mon site web personnel par passion. La première version, simple et basique, combinait principalement HTML avec un peu de PHP, peu de CSS et pas de JavaScript. Pendant mon séjour à Milan, j'ai décidé d'améliorer mon site et c'est ainsi que les deuxième et troisième verions ont vu le jour.

Approche

La première version avait pour but de me familiariser avec le développement web. Pour cela la majeur partie du code fut basée sur du HTML, PHP et CSS. Ensuite, pendant mon séjour à Milan, j'ai relevé le défi de reconstruire entièrement mon site en un week-end afin de le rendre plus moderne. J'ai utilisé un constructeur de site pour créer une maquette le premier jour, puis j'ai (ré)appris les bases du développement web via OpenClassrooms. Le deuxième jour, j'ai recodé intégralement le site, donnant naissance à sa seconde version. Cette seconde version ne contient plus de PHP mais contient plus de CSS ainsi que du JavaScript afin de gérer les langues d'affichage et les animations découlant du positionnement de la souris. Le prochain objectif fut d'optimiser le site pour les appareils mobiles en utilisant les media queries. Pour cela, j'ai décidé de recoder une grosse partie du site, donnant lieu à la troisième version sur laquelle vous naviguez actuellement.

The first version, simple and basic, combined mostly HTML with some PHP, little CSS, and no JavaScript. During my stay in Milan, I decided to improve my site and this is how the second and third versions were born.

Approach

The first version was intended to familiarize me with web development. For this most of the code was based on HTML, PHP and CSS. Then, during my stay in Milan, I took on the challenge of completely rebuilding my site in one weekend to make it more modern. I used a site builder to create a mockup on the first day, then I (re)learned the basics of web development via OpenClassrooms. On the second day, I completely recoded the site, giving birth to its second version. This second version no longer contains PHP but contains more CSS as well as JavaScript in order to manage display languages and animations resulting from mouse positioning. The next goal was to optimize the site for mobile devices using media queries. For this, I decided to recode a large part of the site, giving rise to the third version on which you are currently browsing.
